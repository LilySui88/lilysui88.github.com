## 序
```
时间并不会因为人们的唏嘘而停留，不管高兴也好，难过也罢，时间总是在奔流向前，眨眼间2016走到了尾声。和往年不一样，今年其实我内心上并没有准备好如何给2016来个好好的总结，因为方方面面还有好多不满意又比较无头序的状况，但逃避总不能解决问题，那么就让我好好剖析，认真的回顾一下，给2017年来个警醒吧~

### 关于工作
```
2016年我又换了东家。

虽然很舍不得离开lagou，但毕竟直接领导更换，工作方向、技术氛围不再是自己喜欢的了，那么该走就走，当机立断，来到了现在的公司iqiyi。

iqiyi对于当时的情况来说应该是我能选的最好的选择。这里技术壁垒成熟，各类工具完备，需求规范，工作内容集中不零散，平台优势明显，同事间关系轻松融洽，福利待遇完善，等等方方面面会让人很容易融入进来。从这些方面来说，来到这里是2016年的一个大变化，也是个不错的变化。

在我的岗位位置，它的缺点则是大公司的通病，每个人负责的内容基本就是一两个业务线，并且底层pingback数据仓库和调度、元数据等工具系统都是在其他小组，而我的工作则是相对重复的业务开发，开发工具hive、shell、sqoop几个差不多涵盖了，最近一个月由于报表系统开发的人手不够，我这又加了些php的前端开发，算是又学了点新技术和框架。不过，尽管只是这几个技术点，想要到精通的程度，我还是有一定的差距，不断熟练和精细的过程只能来源于业务开发，so对于现在的情况，我只能继续，不过我希望这个精进的过程不要持续太久，那样就偏离了我最开始想要去的方向。毕竟方向是最重要的，如果方向偏了学的深浅都是无意义。
```

### 关于小家庭
```
2016年，我和冬哥走进婚姻殿堂，有了自己的小家庭。

4月10号求婚（冬哥是被逼的，😁），4月22号领证（紧张、兴奋），5月22号婚礼（温馨甜蜜），一些列的日子都是我们两个一起选的，希望我们彼此能铭记这些美好的瞬间。

2016年，我们有不少吵架的时候，有不少幸福甜蜜的时候，有不少觉得彼此是最好的存在的时候，和顺、摩擦，经历了不少，我们两个也磨合了不少，更加了解彼此的脾气~~~ 每当生病、困难来临，让我们更加感受到，身边多个人陪伴是多么幸福的事情。真的应那句话：陪伴才是最长情的告白~

其实，和一辈子比起来，我们一起走过的路还不长，未来的路难免还会有磕磕绊绊，舌头和牙齿还有打架的时候，更何况是两个性格迥异的大活人呢！！！回忆这一年，自己有不温和的、不该发脾气生气的地方，尽量改掉之~~做的好的地方，继续发扬光大~~~

家和万事兴，深以为然~
```

### 关于旅行
```
2016年的蜜月，我们去了期盼已久的四川、重庆。5月23、24，锦里、都江堰、青城山；5月25，黄龙；5月26，九寨沟；5月27，千古情；5月28，岷江源、到重庆；5月29，朝天门、一棵树观景台、长江索道；5月30，洪崖洞。这8天，每天都过的开心、充实，尝着小吃，看着美景，逛着古迹。这一路上我们相互照顾，感觉到了前所未有的默契。都说旅行能看出两个人的契合度，其实出发前我是忐忑的，毕竟是第一次一起走这么远，担心有变、中间有冲突等，然而事实是我的担心是多余的，这让我也更加确定了这次旅行的意义。

清幽的青城山、九寨沟清澈见底的海子，有时还会浮现在眼前。冬哥说，去了九寨沟，再也对其他山提不起劲儿，因为那实在太美了！！！牦牛肉、青稞酒、豌杂面、重庆老火锅、串串，这些美食都时常让我回忆。我两说好，等有了宝宝一定要再去一次四川和重庆，并且是在深秋，感受那里童话般的秋天~

2016年中秋节，去了心心念念的长白山。虽然我是吉林人，家离那里的距离不过三四百公里，从小就知道长白山，但是我却没有去过。中秋节再也忍不住，终于如愿去了。然而天公不作美，整个假期天气都是阴雨连绵，万幸第一天在西坡的时候，看见了若隐若现的天池，虽然飘着小雨，还有连片的云，但当天池露出脸的那一刻我两都兴奋的不行。第二天，从清晨开始下大雨，知道中午才停，让我们的北坡行，泡了汤，弄的我遗憾不已。然而旅行就是这样，会有不期而遇的惊喜，也有措手不及的无奈，没办法，有缘我还会再去那里，去北坡体验刺激的越野车，吃温泉煮鸡蛋~

长白山回来顺道去了延边，安静闲适的一个小城，夜晚美美的音乐喷泉，人气爆棚的朝鲜族小吃，撑到两个人走不动。记得在延边西站，坐火车的那天，天蓝蓝的，大朵的云彩飘在天空，西站大大的广场，和朝鲜族风格的高铁站，简直就是新海诚画风，美到爆~

去年的时候，希望2016年有两次长距离旅行，这么看来，圆满完成任务~。

对了，差点忘记十一佳佳婚礼，去了吉林市小转了一圈，吃到了人生中最好吃的烧烤，感觉灰常不错的城市~

ps:2014年就想去的泰山，2015没有去，2016依旧没有去，哎，这棵草，2017我一定要拔掉？？？
```

### 关于运动
```
2016年，5、6、7、8跑步了四个月，跑步时整体感觉不错，虽然体重没有变化，但是精神状态保持的比较好。然而9月份跳槽后，整个人都慵懒了下来，没再跑步过，走路回家过几次，也仅限偶尔。待到春暖花开重新跑起来。

2016年爬的山，明显少了。和冬哥爬过的北宫、青城山，和Akia解决掉了西山公园、妙峰山、坡峰岭。对我来说，今年爬山的量远没有达到过瘾的程度。其实，爬山不只为锻炼身体，而是走上山顶远眺那一刹那开阔无比的视野，和身在山顶、山间的心旷神怡。

2016年，在终于把游泳学会了。参加了iqiyi的游泳俱乐部，经同事的指点，让我终于开窍。现在每次可以游1公里，自己都觉得自己棒棒哒，这应该是今年运动里最让自己觉得有成就的事情~

2016年，来了iqiyi，接拍打过一次全场篮球，也算是过了把瘾~对了，在iqiyi，还练过几次瑜伽。

最近在家里，开始keep，测试了一下运动能力，因为几个月没有持续大运动量的动，现在体能下降的很厉害，体重飙升到了历史峰值，感觉压力山大呀！！！上半年自己好好的体力全被暴食、零食、晚睡给糟践啦~~~这个2017必须重新回归~
```

### 关于学习
```
2016年，在学习这个事情上，我说起来应该是最没底气的。

虽然为了准备面试读过hive、linux等几本书，但还是来得太肤浅。

从年初想学机器学习转方向，到后期看了Akia早起学习跟着练英语口语，再到现在开始学日语（为2017年的日本行来点准备），似乎还没有一件事坚持很久，并拙见成果。不得不说，心血来潮的学习动力和临时的方向感让学习更难持之以恒。

无论工作方向技术的学习，还是从休闲兴趣出发的学习，我都应该给自己个小计划和小预期，把时间划分好，在此立FLAG，完成这个总结再出个年度的学习规划表。

都说学习贵在坚持，但是坚持真的真的tmd很贵~~
```

### 关于阅读和写作
```
2016年，我进步了点，读了本《徒步进藏》，更是被书深深鼓舞，深信没有什么不可能。只是书架上是还躺着多本买来就读了几十页的书，这点也是够忏愧。虽然用微信碎片时间也看过一些短文，但质量明显是没法和完整出版的书相比的。所以2017应该多看些出版的书籍，填充大脑，不然长此以往脑袋变草包。

写作从小就不是我擅长的东西，甚至是有点怯。但是，2016年我看到了太多因为写作而在思维体系和方法论得到提升的例子。哪怕只是写几篇口水blog，2017我也希望自己多写点东西。
